Notes & limitations

  • This script controls only audio/video elements on the same page. It cannot change system volume or other tabs’ audio.
  • Some media players use their own JavaScript and may override volume settings. The script attempts to set `element.volume` and also offers a Web Audio fallback for elements that allow it.
